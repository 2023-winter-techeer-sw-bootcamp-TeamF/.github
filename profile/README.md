# 👩‍💻 Instruction
### TAIROT은 AI를 활용한 타로 심리 상담 서비스입니다.
### 오늘의 운세, 연애운, 우정운, 재물운, 소망운 중 하나를 선택해서 여러분의 고민을 나눠주세요.
### AI 타로 마스터가 여러분이 뽑은 카드와 고민을 종합적으로 분석하여 고민 해결에 도움을 드려요.

[미디움 주소](https://localhost)
